# Histórico

Esta pasta preserva documentação antiga, relatórios temporários e auditorias anteriores.

## Pastas

- `originais/`: documentos que ficavam na raiz antes da consolidação em `docs/`.
- `relatorios/`: relatórios operacionais e auditorias temporárias.

## Política

- Não usar esta pasta como documentação principal.
- Consultar os arquivos daqui apenas para rastrear decisões antigas ou recuperar detalhes removidos da documentação consolidada.
- Novos relatórios temporários devem ser arquivados aqui ou em `backups/relatorios/`, nunca na raiz.
